# Point Group Animation Notes

The first implementation targets three prototype scenes:

- `PointGroup4`: tetragonal `4`, a simple four-fold rotation.
- `PointGroupMMM`: orthorhombic `mmm`, combining 2-fold axes, mirror planes,
  and inversion.
- `PointGroup432`: cubic `432`, showing 4-fold, 3-fold, and 2-fold axes.

The 32-point-group data table is present in `pointgroup/data.py`, but most
groups still need visual tuning after the prototype scenes are rendered.
